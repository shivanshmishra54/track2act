# Track2Act — Full Stack

## Folder Structure
```
track2act/
├── frontend/   → React + Vite (port 5173)
└── backend/    → Python + FastAPI + SQLite (port 8000)
```

## How to Run

### Step 1 — Start the Backend
```
cd backend
pip install -r requirements.txt
python main.py
```

### Step 2 — Start the Frontend
```
cd frontend
npm install
npm run dev
```

Open http://localhost:5173
